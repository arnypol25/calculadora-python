# Calculadora

Calculadora con interfaz gráfica desarrollada en Python usando Tkinter.

## Funcionalidades
- Operaciones básicas: suma, resta, multiplicación, división
- Potencias (x^y)
- Gestión de errores (división entre cero, resultados demasiado largos, entradas inválidas)
- Interfaz centrada automáticamente en pantalla

## Tecnologías
- Python
- Tkinter (interfaz gráfica)

## Cómo ejecutarlo
```bash
python "Calculadora.py"
```
